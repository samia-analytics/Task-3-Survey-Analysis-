# 📊 Survey Data Analysis

## 📌 Objective
Analyze user satisfaction survey data and visualize the results.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Features
- Created survey dataset
- Counted responses using `value_counts()`
- Visualized results using a bar chart

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Run all cells step-by-step

## 📌 Learning Outcome
- Understanding survey data analysis
- Using `value_counts()` for counting
- Creating bar charts for visualization
